Develop scripts encapsulation and dependencies collection.

## Install

`$ npm i @akijoey/devkit -D`

## Usage

Lint script.

`$ devkit lint`

Test script.

`$ devkit test`

Format script.

`$ devkit format`

Prepare script.

`$ devkit prepare`
